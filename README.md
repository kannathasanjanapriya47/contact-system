# Contact Form Email System

## Description
This project allows users to submit queries via an HTML form.

## Features
- Sends query details to admin email
- Sends confirmation email to client
- Uses SMTP for reliable email delivery

## Technologies Used
- HTML
- PHP
- PHPMailer
- Gmail SMTP

## How to Run
1. Install XAMPP
2. Place project inside htdocs folder
3. Start Apache server
4. Configure email and app password in send.php
5. Open browser and go to:
   http://localhost/contact-system/

## Output
- Admin receives query email
- Client receives confirmation email
